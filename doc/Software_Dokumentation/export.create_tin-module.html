Create a triangulated irregular network.

The triangulated irregular network is created in the output directory.

Parameters:
  • in_tin_points(DEFeatureClass) - The input points which are used to create a triangulated irregular network.
  • out_directory(DEFolder) - The output directory.
  • out_tin_name(GPString) - The name of the output triangulated irregular network.
  • triangulation_technique(GPStrin) - The triangulation technique used along breaklines.
