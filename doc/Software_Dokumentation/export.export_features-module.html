export_features_to_feature_class(in_features, out_path, out_name)

source code 

Export the input features to a feature class.

The feature class is created in a new file geodatabase.

Parameters:
  • in_features(DEFeatureClass) - The input features which are exported to a feature class.
  • out_path(DEFolder) - The output path.
  • out_name(GPString) - The name of the output file.

export_features_to_esri_shape(in_features, out_path, out_name)

source code 

Export the input features to an esri shapefile.

After creating an output folder 'esri_shape' the function FeatureClassToFeatureClass_conversion is executed. The output file is a shapefile, because there is a normal file and not a file geodatabase specified as output folder.

Parameters:
  • in_features(DEFeatureClass) - The input features which are exported to an esri shapefile.
  • out_path(DEFolder) - The output path.
  • out_name(GPString) - The name of the output file.

export_features_to_ascii_xyz(in_features, out_path, out_name, fields)

source code 

Export the input features to an ascii xyz file.

Therefore an output folder 'ascii' is created. A temporary feature class is created to display the coordinates. Then the function ExportXYv_stats is called and executed. Finally the temporary feature class is deleted.

Parameters:
  • in_features(DEFeatureClass) - The input features which are exported to an ascii xyz file.
  • out_path(DEFolder) - The output path.
  • out_name(GPString) - The name of the output file.
  • fields(GPString) - The field names of the input file.

export_features_to_autocad(in_features, out_path, out_name, autocad_type)

source code 

Export the input features to an autocad file.

First, an output folder 'autocad' is created. Then the function ExportCAD_conversion is called and executed.

Parameters:
  • in_features(DEFeatureClass) - The input features which are exported to an autocad file.
  • out_path(DEFolder) - The output path.
  • out_name(GPString) - The name of the output file.
  • autocad, type(GPString) - The autocad file version.
