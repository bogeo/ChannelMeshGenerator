Package preparation

source code


Created on 17.07.2017

@author: Matthias Hensen

Package with modules and functions to generate and adjust input data.

The functions and parameters are explained in the respective modules.

module bound_features_to_investigated_area:
    Bound the input features to the investigated area.
module connect_cross_sections:
    Connect the points from the cross sections to lines.
module create_stream_center_line:
    Create the center line of a stream.
module create_stream_polygon:
    Create the investigated area as polygon.
module flip_line:
    Check and flip line directions or flip line numerations.
module load_points:
    Load points from a csv, txt or xyz file to a feature class.
module subdivide_water_land_border:
    Subdivide the water land border at the cross sections.
